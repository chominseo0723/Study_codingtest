# 지폐 접기
   - 지폐를 접을 때는 항상 길이가 긴 쪽을 반으로 접습니다.
   - 접기 전 길이가 홀수였다면 접은 후 소수점 이하는 버립니다.
   - 접힌 지폐를 그대로 또는 90도 돌려서 지갑에 넣을 수 있다면 그만 접습니다.
   - 해당 조건을 만족하는 배열 2개가 매개변수로 주어집니다.
