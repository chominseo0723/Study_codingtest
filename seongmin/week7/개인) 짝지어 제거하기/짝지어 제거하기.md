# 짝지어 제거하기
   - 해당 문제는 두개의 같은 알파벳이 붙어있는 짝을 찾고 찾는다면 제거합니다.
   - 위 과정을 반복해서 문자를 모두 없앨 수 있는지 확인해야하는 문제입니다.
## 알고리즘

1. 스택
   - 간단한 스택으로 구현이 가능합니다. 빈 스택을 하나 선언하고 주어진 문자열을 돌면서
   - 스택이 비었다면 문자를 삽입하고 스택이 비어있지 않으면서 문자가 같은 경우에는 스택에 집어넣습니다.
   - 최종적으로 스택이 비어있다면 모두 제거가 된것이므로 1을 리턴하고 스택이 비어있지 않다면 모두 제거가
   - 불가능하다는 소리이므로 0을 리턴합니다.


2. 결과 도출
   - 최종적인 결과를 리턴합니다.
